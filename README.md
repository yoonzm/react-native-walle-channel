# react-native-walle-channel

美团多渠道打包插件 react-native sdk

## Installation

```sh
npm install react-native-walle-channel
```

## Usage

```js
import WalleChannel from "react-native-walle-channel";

// ...
WalleChannel.getChannel();
WalleChannel.getExtraInfo();
```
